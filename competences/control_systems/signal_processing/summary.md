# Statistical Signal Processing

Full course at master's level at SDU Sønderborg covering estimation theory
and filtering from first principles up to Kalman filtering.

## Topics Covered

**Classical Estimation:**
- Probability recap and foundations
- Least Squares (LS) and Recursive Least Squares (RLS)
- Maximum Likelihood Estimation (MLE)
- Maximum A Posteriori (MAP)
- Minimum Mean Square Error (MMSE)
- Linear MMSE (LMMSE)
- Estimator concepts — bias, consistency, efficiency

**Kalman Filtering:**
- Kalman Filter (KF) — linear systems
- Extended Kalman Filter (EKF) — nonlinear systems

## Practical Application
Theory from this course was directly applied in the Amfitech in-company project
where a sequential Extended Kalman Filter was implemented for magnetic tracking.
See `competences/control_systems/kalman_filtering/summary.md` for project detail.

## Honest Framing
This was a rigorous theoretical course. Practical implementation experience
