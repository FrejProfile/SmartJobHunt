# Software Design

## Hong Kong University of Science and Technology
Exchange semester course in software engineering covering UML modelling
and design patterns. Built a Java application as the course project
including unit testing. Focus on structured software design rather than
just implementation.

## SDU — Software for Embedded Systems
Embedded software architecture and design, data types relevant to embedded
contexts — fixed point, bit width considerations, memory layout.

## SDU — Embedded Software Design
Further embedded software design principles, building on the above.
Overlap with hardware/software co-design thinking from FPGA courses.

## Honest Framing
Java is not a primary language and not relevant for most target roles.
The value here is the structured design thinking — UML, design patterns
and understanding how software architecture decisions affect embedded systems.
