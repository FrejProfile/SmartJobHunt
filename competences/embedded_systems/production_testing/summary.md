# Production Testing

## Swienty A/S, Sønderborg — Student Worker (2021-2022)
Built and tested Electrical Control Units for industrial bee production equipment.

## The Work
Assembly and functional testing of ECUs before shipping to customers.
Units had to pass defined functional test requirements — not shipped until passing.
Real production environment with quality requirements, not a student project.

PLCs were implemented in many of the designs. Work included uploading PLC programs
to the units and verifying correct behaviour as part of the functional test process.
Not responsible for writing the PLC programs but hands-on with uploading,
configuring and testing them in a real production context.

## Honest Framing
This was a student worker position, not a graduate engineering role.
The value is real exposure to a production environment with functional
test requirements, quality gates and PLC based control systems.
Do not overclaim seniority but represent it as genuine industry experience
with real pass/fail criteria.
